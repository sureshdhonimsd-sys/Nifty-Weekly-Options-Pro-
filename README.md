Professional TradingView Pine Script for Nifty Weekly Options.

- Opening Range Breakout (ORB)
- EMA 9 / EMA 20
- VWAP Filter
- Supertrend Confirmation
- Volume Confirmation
- ATR Stop Loss
- Risk Reward Target
- Buy / Sell Alerts
- Dashboard
